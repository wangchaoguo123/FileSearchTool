```# 本地文件搜索工具

## 项目概述

这是一个基于 PyQt5 开发的本地文件搜索工具，提供了友好的图形用户界面，支持按关键词搜索文件，具备文件预览、结果排序、CSV导出等功能。该工具旨在帮助用户快速定位本地计算机上的特定文件。

### 主要功能特性

- **多类型文件搜索**：支持按文件名关键词搜索各种类型的文件
- **文件类型过滤**：预设多种文件类型过滤器，支持自定义文件类型
- **实时预览**：支持文本文件和图片文件的实时预览
- **结果排序**：支持按名称、修改时间、文件大小等多种方式排序
- **进度显示**：实时显示搜索进度
- **结果导出**：支持将搜索结果导出为CSV文件
- **美观界面**：现代化的UI设计，支持深浅主题切换
```

## 使用说明

### 启动程序
运行 `python main.py` 启动程序，将看到主界面窗口。

### 基本操作步骤
1. **选择搜索路径**：点击"浏览文件夹"按钮选择要搜索的目录
2. **输入关键词**：在关键词输入框中输入要搜索的文件名关键词
3. **设置文件类型过滤**（可选）：
   - 勾选预设的文件类型复选框
   - 或在自定义类型框中输入扩展名（如 `.txt,.docx`）
4. **开始搜索**：点击"开始搜索"按钮
5. **查看结果**：搜索结果显示在下方列表中
6. **预览文件**：点击结果列表中的文件，在右侧预览区域查看
7. **排序结果**：使用"排序方式"下拉框调整结果排序
8. **导出结果**：点击"导出CSV"按钮保存搜索结果

### 高级功能
- **实时进度**：搜索过程中显示进度条
- **停止搜索**：随时点击"停止搜索"按钮中断搜索
- **双击打开**：双击搜索结果直接用默认程序打开文件
- **多种排序**：支持按名称、时间、大小等多种方式排序

## 示例

### 示例 1：搜索 PDF 文档
1. 点击"浏览文件夹"选择文档目录，如 `D:\Documents`
2. 在关键词框输入 `报告`
3. 勾选"Office文档"复选框（包含PDF）
4. 点击"开始搜索"
5. 程序将显示所有包含"报告"的PDF文档

### 示例 2：搜索图片文件
1. 选择图片存放目录，如 `C:\Users\用户名\Pictures`
2. 关键词输入 `生日`
3. 勾选"图片文件"复选框
4. 点击"开始搜索"
5. 在结果列表中点击任意图片即可在右侧预览
6. 可使用排序功能按修改时间查找最新照片
